Long Read Allele Specific analysis¶

LoRALS is a package and collection of tools for conducting allele-specific analyses in Oxford Nanopore-generated long read data

Contents:

  • Command-Line Programs Listing

Installation¶

LoRALS is available on GitHub at https://github.com/mojaveazure/lorals and can be installed with pip

pip install git+https://github.com/mojaveazure/lorals.git
